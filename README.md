# KidsChores (Modified Fork)

A Home Assistant custom integration for managing kids' chores, allowances, and rewards.

This is a modified fork of [kidschores-ha](https://github.com/ad-ha/kidschores-ha) by [@ad-ha](https://github.com/ad-ha).

## Changes from Original

- Custom chore assignments for family needs
- Modified button behaviors
- Adjusted sensor configurations

## Installation

### Manual Installation

1. Copy the `custom_components/kidschores` folder to your Home Assistant `custom_components` directory
2. Restart Home Assistant
3. Configure via UI: Settings → Devices & Services → Add Integration → KidsChores
